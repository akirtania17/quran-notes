# Tester Onboarding (EAS Dev Build)

## Install
- You’ll receive an EAS internal/dev build link. Install on your device (iOS/Android).
- The app is pre-configured to hit the production backend; no Settings changes needed.

## What to try
1) Record a short clip (30–60s) and upload.  
2) Watch processing; if it stalls, pull-to-refresh or tap “Start Processing.”  
3) Open the finished session, read transcript/translation/summary.  
4) Add a note.  
5) Bookmark the session and confirm it appears under Bookmarked on Home.  
6) Add a highlight if applicable.

## Reporting issues
- Open Settings → Diagnostics and run “Check health.”
- Include a screenshot of Diagnostics plus the error you saw.
- Note the time and what you were doing (recording, uploading, refreshing, etc.).
- If an error shows `X-Request-Id`, include it.
