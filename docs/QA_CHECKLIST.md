# QA Checklist

- Devices: at least 1 iOS + 1 Android (physical), plus emulator/simulator sanity.
- Record lengths: 1m, 5m, 15m; confirm upload and processing complete.
- Network: toggle Wi‑Fi off during upload (expect retry UI); retry after restoring network.
- Processing restart: restart backend, then tap “Start Processing” on an Uploaded session.
- Tabs: Transcript/Translation/Summary/Notes render without crashes on empty content.
- Notes: add note (+1 XP), retry note load failure, ensure notes list renders.
- Bookmarks/highlights: toggle bookmark; add/remove highlight; verify Home filter shows bookmarked.
- XP/streak: upload (+5), open a complete session (+2) once per session/day, note (+1); streak increments once/day.
