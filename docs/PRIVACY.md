# Privacy (MVP)

- No accounts; the app uses an anonymous device ID stored locally.
- Sessions: audio is sent to the backend for transcription/translation/summarization; outputs are stored with the session.
- Notes/highlights/bookmarks are tied to the device ID.
- XP and streaks are stored locally on the device (cleared if you reinstall or reset local data).
- For tester builds, the backend URL is baked into the app; you can view it under Settings → Diagnostics.
